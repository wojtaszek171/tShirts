The Model Targets (advanced recognition) sample demonstrates the detection and tracking of two example objects in arbitrary order.

Key Functionality

  • Loading and activation of two Model Targets (each having two Detection Positions) at the same time
  • Detection of the object in camera view
  • Automatic 3D object tracking after successful detection
  • Device Tracker when the object is not visible in the camera view
  • Manage camera functions: continuous autofocus

Physical Targets

  • Model Target: Mars Lander 3D Printed Model, two views
  • Model Target: Motorcycle Toy, two views

Instructions

  • Point camera at one of the two objects (Viking Lander or the KTM Bike)
  • Move around the object
  • Point the camera to the next object
  • Move around the object
  • Single tap to focus
  • Double tap to access the options menu
  • Reset to start over the experience
